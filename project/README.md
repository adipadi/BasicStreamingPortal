## Installing project
1. Install node
2. Run npm install
3. Checkout branch oppgave_1_b (run the command git checkout oppgave_1_b)
3. In one terminal window: npm run dev
4. In a second terminal window: npm run start-backend
5. Visit http://localhost:8080
